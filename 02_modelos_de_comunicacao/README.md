# 02 — Modelos de Comunicação: OSI e TCP/IP

Este diretório apresenta os modelos conceituais usados para descrever como sistemas se comunicam em rede:

- Modelo OSI (7 camadas)
- Modelo TCP/IP (4 camadas)
- Comparações entre os modelos
- Relação entre camadas e protocolos

## Objetivo didático
Compreender como os modelos organizam a comunicação em camadas e como isso facilita o projeto, análise e depuração de redes e aplicações distribuídas.
