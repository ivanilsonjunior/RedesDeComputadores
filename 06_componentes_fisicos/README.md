# 06 — Componentes Físicos de Redes

Materiais sobre dispositivos e meios físicos utilizados em redes de computadores.

## Conteúdo
- Placa de Rede (NIC)
- Hubs, Switches e Roteadores
- Repetidores
- Access Point
- Cabos (UTP, STP, fibra óptica)
- Rádio (Wi-Fi, LoRa, Bluetooth)

## Objetivo didático
Compreender o papel dos dispositivos físicos na rede e como eles afetam a comunicação lógica.
