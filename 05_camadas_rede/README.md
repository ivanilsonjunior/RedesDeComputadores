# 05 — Camada de Rede

Este diretório contém scripts didáticos relacionados a endereçamento IP, encaminhamento e conceitos básicos de protocolo IP.

## Exemplos disponíveis
- `ping_simplificado_python3.py`  
  Simula o comportamento de um "ping" sem usar ICMP raw.

## Objetivo didático
Mostra os conceitos de:
- TTL
- Encaminhamento
- Latência e RTT
- Porta vs Endereço IP
