# 01 — Conceitos Básicos de Redes

Este diretório contém materiais introdutórios sobre os principais conceitos de Redes de Computadores, incluindo:

- O que é uma rede
- Tipos de topologias (barramento, estrela, anel, malha)
- Tipos de redes (PAN, LAN, MAN, WAN)
- Noções de comunicação de dados
- Legislação e regulamentações básicas ligadas à Internet no Brasil

## Objetivo didático
Oferecer uma visão geral sobre os elementos fundamentais que formam qualquer rede, preparando o aluno para entender conceitos mais avançados como protocolos, camadas e modelos.
